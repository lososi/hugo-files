## Что это и зачем?

Тесткит - набор специальных реактивов, которые при контакте с разными веществами дают разные цвета. По совокупности цветов, который дает каждый из реактивов можно с хорошей точностью определить вещество.
Применяется как одна из практик снижения вреда.
